Final University Project

Multiple Encryption Tool

The primary goal of this project is to employ two different encryption algorithms into a Graphical User Interface. However, in order to reach this goal, thorough research and planning will be undertaken along the way. This project will explore different encryption algorithms used in the current modern era of technology. The research completed into the most secure encryption algorithms will then be applied to create an Encryption Tool. This Encryption Tool will include two different encryption algorithms; which a user will be able to select in order to encrypt and decrypt a file. A Graphical User Interface shall also be developed in order to employ the encryption algorithms. The GUI will be designed in such a manner to be as user-friendly as possible so that users can easily encrypt or decrypt documents as they please.
